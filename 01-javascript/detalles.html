Ingeniero de Software

Tech Solution Inc. | Remoto

Descripción del Puesto

Buscamos un ingeniero de software con experiencia en desarrollo web y conocimientos en JavaScript, React y Node.js. El candidato ideal debe ser capaz de trabajar en equipo y tener buenas habilidades de comunicación.

Responsabilidades

  • Desarrollar y mantener aplicaciones web.
  • Colaborar con el equipo de diseño para crear interfaces de usuario atractivas.
  • Participar en revisiones de código y asegurar la calidad del software.

Requisitos

  • Experiencia demostrable en desarrollo web.
  • Conocimientos sólidos de JavaScript, React y Node.js.
  • Habilidades de resolución de problemas y pensamiento crítico.

Acerca de la empresa

Tech Solution Inc. es una empresa líder en el desarrollo de soluciones tecnológicas innovadoras. Nos especializamos en el desarrollo web y móvil, y trabajamos con las últimas tecnologías para ofrecer productos de alta calidad a nuestros clientes.
